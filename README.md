# PMUG

Algoritmo desenvolvido durante meu Trabalho de Conclusão de Curso no Bacharelado em Engenharia Elétrica, pela Universidade Federal de São João del-Rei.

Nele, foi desenvolvido um algoritmo de otimização multiobjetivo, conhecido como NSGA-II, para o problema da manutenção de unidades geradoras, ou PMUG. Resumidamente, o algoritmo tem a função de determinar qual o melhor período, dentro de um ano, para a entrada em manutenção de oito unidades geradoras de um sistema elétrico de potência. Na realização dessa otimização, três objetivos são considerados: receita média do produtor, índice de confiabilidade ligado a energia esperada não suprida durante um ano e custo médio de produção de energia.
